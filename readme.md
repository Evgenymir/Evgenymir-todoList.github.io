## Для работы надо установить
* nodejs
* npm
* webpack

## Устанавливаем зависимости
* npm install

## Старт работы
* npm run dev

Поднимается локальный сервер - `localhost:3001`

Страницы доступны - `localhost:3001/`

## Структура папок
Все рабочие файлы лежат в папке `src`
Картинки, шрифты лежат в папке `public`
